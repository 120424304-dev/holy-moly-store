# CSV 文件保存步骤详解

## Windows 系统

### 方法1: 记事本 (推荐)

1. **打开记事本**
   - 按 `Win + R`，输入 `notepad`，回车
   - 或开始菜单搜索 "记事本"

2. **粘贴内容**
   - 把我发的 CSV 内容全部复制
   - 在记事本里按 `Ctrl + V` 粘贴

3. **另存为 CSV**
   - 点击菜单: `文件` → `另存为`
   - 选择保存位置 (比如桌面)
   - **文件名** 输入: `holy-moly-products-usd.csv`
   - **保存类型** 选择: `所有文件 (*.*)`
   - **编码** 选择: `UTF-8`
   - 点击 `保存`

4. **验证文件**
   - 保存后，文件图标应该是 Excel 图标
   - 右键文件 → 属性，确认后缀是 `.csv`

---

### 方法2: Excel (查看方便)

1. **打开记事本**，粘贴 CSV 内容
2. **另存为** → `holy-moly-products-usd.txt` (先保存为txt)
3. **打开 Excel**
4. `数据` → `从文本/CSV` → 选择刚才的 txt 文件
5. 确认分隔符是 `逗号`，编码是 `UTF-8`
6. `文件` → `另存为` → 选择 `CSV UTF-8 (逗号分隔)`

---

## Mac 系统

### 方法1: 文本编辑

1. **打开"文本编辑"** (在 应用程序 → 实用工具)
2. **粘贴** CSV 内容
3. **格式** → **制作纯文本** (必须！)
4. `文件` → `存储`
5. **文件名**: `holy-moly-products-usd.csv`
6. **编码**: `Unicode (UTF-8)`
7. 点击 `存储`

### 方法2: 终端命令

```bash
# 创建文件并粘贴内容
cat > ~/Desktop/holy-moly-products-usd.csv << 'EOF'
[粘贴CSV内容到这里]
EOF
```

## 常见问题

### ❌ 文件变成 `holy-moly-products-usd.csv.txt`
**原因**: Windows 隐藏了扩展名
**解决**:
1. 文件资源管理器 → `查看` → 勾选 `文件扩展名`
2. 重命名文件，删除 `.txt`

### ❌ 中文显示乱码
**原因**: 编码不是 UTF-8
**解决**: 重新保存，确保选择 `UTF-8` 编码

### ❌ Shopify 导入失败
**检查清单**:
- [ ] 文件后缀是 `.csv`
- [ ] 编码是 UTF-8
- [ ] 文件大小不超过 15MB
- [ ] 列名没有被修改

## 验证文件是否正确

### 用 Excel 打开检查
1. 双击 CSV 文件
2. 检查:
   - 第一行是列名 (Handle, Title, Body...)
   - 第二行开始是产品数据
   - 中文显示正常，没有乱码
   - 价格列显示美元 ($23.47 等)

### 用文本编辑器检查
1. 右键 CSV 文件 → 打开方式 → 记事本
2. 检查:
   - 内容是用逗号 `,` 分隔的
   - 包含 `"` 引号包裹的文本
   - 没有奇怪的符号

## 下一步: 上传到 Shopify

1. 登录 Shopify 后台
2. `Products` → `Import`
3. 点击 `Add file`，选择保存的 CSV
4. 点击 `Upload and preview`
5. 确认无误后点击 `Import products`
