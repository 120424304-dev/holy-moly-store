## 批量上传技巧

### 方法一: Shopify 后台批量上传
1. Products → 勾选多个产品 → Actions → Add images
2. 按命名排序后批量选择

### 方法二: CSV 更新图片链接
```csv
Handle,Image Src
holy-moly-beginner-120,https://你的CDN.com/holy-moly-beginner-120-01.jpg
holy-moly-beginner-120,https://你的CDN.com/holy-moly-beginner-120-02.jpg
```

### 方法三: 第三方工具
- **Matrixify** (原 Excelify) - 适合大量图片
- **Bulk Image Edit** - Shopify App Store

## 图片规范建议

| 项目 | 建议 |
|-----|------|
| 尺寸 | 2048 x 2048 px (正方形) |
| 格式 | JPG 或 WebP |
| 大小 | 单张 < 20MB |
| 背景 | 纯白或浅灰 |
| 首张 | 白底产品图 |
| 次张 | 使用场景/细节 |
